# 线性表
        线性表是最简单最常用的一种数据结构。简言之，一个线性表是n个数据元素的有序列表（集合内的元素类型一致）。
    至于每个元素的具体含义，在不同情况下各不相同，它可以是一个数字或者一个符号，也可以是一页书，甚至更复杂的东西。
